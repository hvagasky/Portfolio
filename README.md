# Portfolio
This repository contains examples of scripts, presentations, and publications created by Hannah Vagasky since 2024. 

In addition to this repository, https://github.com/hvagasky/HAILMELT_2024manuscript is a public repository that contains a Python based model that simulates falling, melting hail that was created by Hannah and is associated with her publication entitled "The Sensitivity of Falling Hail to Different Melting and Terminal Velocity Parameterizations and Environmental Conditions" (https://journals.ametsoc.org/view/journals/mwre/153/11/MWR-D-24-0180.1.xml) and with the presentation included in this repository called "Hail_Seminar2025.pptx".
